
Verilog/SystemVerilog document generator based on doxygen

Run cmake with the makefile generator

create direcory 

mkdir build
cd build

Run cmake with the makefile generator
cmake -G "Unix Makefiles" ../sysverilog_antlr4_parser

for further information see https://www.doxygen.nl/manual/index.html

You can find some examples in directory sysverilog_antlr4_parser/sysverilog_examples
